Provider Configuration

Option Type Description
autoInsert boolean Automatically insert default rules on initialization.
defaultRules SpeculationRules Default speculation rules to apply.

Service Methods

Method Return Type Description
insertRules(rules, id?) string | null Insert speculation rules into the document. Returns the script ID.
removeRules(id) boolean Remove previously inserted rules by ID.
clearAllRules() void Remove all speculation rules from the document.

Service Properties

Property Type Description
isSupported SpeculationSupport Check browser support for Speculation Rules API.
isPrerendering boolean Check if the document is currently prerendering.

Eagerness Levels

Level Description
immediate Speculate as soon as possible.
eager Speculate on any small positive signal.
moderate Speculate on hover for 200ms.
conservative Speculate only on pointer/touch down.
