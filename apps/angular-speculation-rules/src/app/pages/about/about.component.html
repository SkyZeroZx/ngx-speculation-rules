What is Speculation Rules API?

The Speculation Rules API is a browser feature that allows developers to hint which navigations should be prefetched or prerendered. This can significantly improve perceived performance by loading resources before the user actually clicks on a link.

Why ngx-speculation-rules?

  • Easy integration with Angular applications
  • SSR compatible
  • Zoneless mode ready
  • Type-safe configuration
  • Dynamic rule management
